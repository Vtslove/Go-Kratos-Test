# 博客后台管理系统

### 技术栈

- [TypeScript](https://www.typescriptlang.org/)
- [Es6+](http://es6.ruanyifeng.com/)
- [Vite](https://vitejs.dev/)
- [Vue3](https://v3.vuejs.org/)
- [Pinia](https://pinia.vuejs.org/)
- [Ant-Design-Vue](https://2x.antdv.com/docs/vue/introduce-cn/)

## 安装使用

- 安装依赖库

```bash
pnpm install
```

- 运行

```bash
pnpm serve
```

- 编译

```bash
pnpm build
```
